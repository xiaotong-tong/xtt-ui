Text Edit 输入框 ​

输入框，用于替代原生的 textarea 元素，提供丰富的样式和功能。

基础用法 ​

This is text-edit
html
<xtt-text-edit>This is textarea</xtt-text-edit>

block ​

将输入框转换为块级元素。

html
<xtt-text-edit block></xtt-text-edit>

rows ​

设置输入框的行数。（显示的高度）

html
<xtt-text-edit rows="1"></xtt-text-edit>

<xtt-text-edit rows="5"></xtt-text-edit>
