Input 输入框 ​

输入框，用于替代原生的 input 元素，提供丰富的样式和功能。

基础用法 ​

html
<xtt-input />
<xtt-input placeholder="placeholder" />
<xtt-input value="input value" />
<xtt-input value="readonly value" readonly />
<xtt-input value="disabled value" disabled />

block ​

将输入框转换为块级元素。

html
<xtt-input block />
