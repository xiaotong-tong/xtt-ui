基础用法

This is tooltip default value

添加 data-xtt-tooltip (下文缩写为 data) 属性后, tooltip 显示的内容为 data 内的文字，
但是仍希望在 xtt-tooltip 元素内添加一个默认值, 这个默认值不会影响显示，也不会影响屏幕阅读器读取的内容, 它会使你的无障碍功能更健壮
因为 data 的内容是在 tooltip 显示前才会添加到 xtt-tooltip 元素内的，在静态检查时 tooltip 的内容为空，这会略微影响无障碍检查
